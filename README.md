# User Management API

This is a simple user management API built with Express.js, featuring user registration, login, profile management, and JWT-based authentication.

## Features
- User Registration
- User Login
- Get User Profile
- Update User Profile
- Delete User Profile
- JWT Authentication
- Swagger API Documentation

## Technologies
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Swagger
